# Play-Game-Alex
Write a function `rectangle()` that takes two integers (x, y).  This function should print a rectangle of characters on the screen with:  'o' at the corners '-' on the horizontal edges '|' on the vertical edges ' ' spaces inside

Rules
If x < 1 or y < 1, nothing should be printed.
No infinite loops or errors are allowed.
Only print what is specified (nothing extra).
Do not modify the `main()` function.
Add a `DocString` to the `rectangle()` function.
Use the `sys` module to read command-line arguments.
Functions and variables should be clear and descriptive, and must be in English.
